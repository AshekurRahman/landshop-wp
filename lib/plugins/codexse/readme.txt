== Installation ==

**Step 1:** Upload the plugin file to install by navigating through Plugins➔Add New➔Upload Plugin➔Choose File to Install from your WordPress dashboard or you can simply search for the Codexse plugin from the plugin directory by going to Plugins➔Add New and search for the plugin in the search tab to install it on your WordPress site.

**Step 2:** After successful installation, you have to click the "activate" button to activate the codexse for Elementor.

**Step 3:** When you activate Codexse for Elementor you will be redirected to our CodexseAddons Dashboard Home Tab.

For a more detailed explanation check out the following documentation
